## 技巧细节注意
1. 切片[:]返回原切片,[:0]返回空切片
2. string遍历用for i:=0;i<len(str);i++ 遍历得到的是byte
3. 排序
    ```
    sort.Ints([]int{})
    // 字符串排序
    sort.Strings([]string{})
    // 自定义排序
    sort.Slice(s,func(i,j int)bool{return s[i]<s[j]})
   ```
4. 切片append扩容指针地址会变
